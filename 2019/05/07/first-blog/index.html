Hexo博客搭建（windows）

  1. 安装node.js cmd中 npm install -g hexo -cli
  2. 安装git
  3. 注册github 配置SSH秘钥
  4. 新建repository 在 Repository name 添加以后可以访问的地址（eg:用户名.github.io）
  5. 本地新建文件夹 eg：blog
  6. 新建一个放博客文章的文件夹 eg：anlies
  7. 初始化 hexo init anlies
  8. 进入blog cd anlies
  9. npm install
  10. npm install hexo-deployer-git –save （hexo部署到git上）
  11. hexo g （生成静态文件）
  12. hexo d （提交到github）
