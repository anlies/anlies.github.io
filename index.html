Hexo博客搭建（windows）

  1. 安装node.js cmd中 npm install -g hexo -cli
  2. 安装git
  3. 注册github 配置SSH秘钥
  4. 新建repository 在 Repository name 添加以后可以访问的地址（eg:用户名.github.io）
  5. 本地新建文件夹 eg：blog
  6. 新建一个放博客文章的文件夹 eg：anlies
  7. 初始化 hexo init anlies
  8. 进入blog cd anlies
  9. npm install
  10. npm install hexo-deployer-git –save （hexo部署到git上）
  11. hexo g （生成静态文件）
  12. hexo d （提交到github）

博客编写

markdown：文档结构描述语言，Markdown不是排版工具，而是内容写作工具
markdown编写：（英文状态下）

  • #一级标题
  • ##二级标题
  • ###三级标题
  • ####四级标题
  • #####五级标题
  • > 引用
  • - 无序列表
  • 1. 有序列表
  • ![名字]（图片地址）插入图片
  • 百度 超链接
  • 加粗
  • 居中
  • 设置字体颜色 大小 字号

  • 设置字体方向
